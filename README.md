# DTY Edu-Textile

Website informatif dan edukatif mengenai proses produksi benang **DTY (Draw Textured Yarn)**, ditargetkan untuk masyarakat umum usia SMA ke atas. Proyek ini dibangun menggunakan teknologi web native untuk menjamin kecepatan, kesederhanaan, dan orisinalitas kode.

## ✨ Fitur Utama 
1. **Profil Unit DTY**: Informasi mengenai Unit DTY Politeknik Enjinering Indorama (PEI) yang berdiri sejak 2013 bekerjasama dengan PT Indorama Synthetics.
2. **Alur Produksi Interaktif (8 Tahap)**: Penjelasan urut proses manufaktur dari bahan mentah hingga distribusi lengkap dengan toggle "Detail Teknis" untuk informasi lebih mendalam.
3. **Dokumentasi & Keselamatan Kerja (K3)**: Informasi mengenai praktik lapangan mahasiswa dan standar Alat Pelindung Diri (APD) di area mesin.
4. **Smart Tooltips**: Penjelasan istilah teknis (seperti POY, DTY, Murata, Knitting, Grading) secara instan saat kursor diarahkan (hover) atau diklik pada perangkat mobile.
5. **Kamus Saku (Glosarium)**: Fitur pencarian dan filter istilah teknis tekstil secara real-time di bagian bawah halaman.

## 📂 Struktur Direktori
```text
DTY/
├── Gambar/           # Folder berisi 6 gambar produksi asli
├── index.html        # Halaman utama website
├── style.css         # Styling kustom (responsif & modern)
├── script.js         # Logika interaktif (tooltip, akordeon, glosarium)
└── README.md         # Dokumentasi petunjuk proyek
```

## 🚀 Cara Menjalankan
Cukup buka file `index.html` menggunakan peramban (browser) web apa pun seperti Google Chrome, Microsoft Edge, atau Mozilla Firefox. Tidak diperlukan server lokal atau proses instalasi tambahan.
